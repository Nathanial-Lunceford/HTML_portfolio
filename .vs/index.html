Tacoparser

Given a .CSV file listing latitudes, longitudes and names of Taco Bells in the Southern US, the program finds the two store locations that are the farthest apart. Location and names from .CSV are imported into instances of the TacoBell class, which are then added to an array. Two for loops are run, the second inside the first. The first gets every TacoBell from the array, the second finds the distance between that TacoBell and every other one. The largest distance is printed out. Created test cases for importing latitude and longitude into TacoBell class, the TacoBell uses an interface for ease of use.
